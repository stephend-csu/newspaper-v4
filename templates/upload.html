Address APIs Overview

1. OpenStreetMap Nominatim & Census Geocoder API
Resolves delivery street addresses against official public geographic databases to determine geographic coordinates (latitude and longitude) and verify candidate cities in Contra Costa County, CA.
2. OSRM (Open Source Routing Machine) API
Computes real-world road network driving distances between delivery addresses, solves Traveling Salesperson Problem (TSP) address ordering starting from 2505 Dean Lesher Dr, Concord, CA, and calculates driving leg miles between addresses.
